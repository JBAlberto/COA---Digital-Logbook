## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/JBAlberto/COA---Digital-Logbook.git
cd REPOSITORY
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
